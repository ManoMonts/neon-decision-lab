# Neon Decision Lab

Um mini jogo gratuito de navegador sobre decisões rápidas, risco e combo.

A ideia é simples: cada carta apresenta uma situação caótica e você precisa escolher uma resposta. Cada escolha altera sua pontuação, risco e combo. O objetivo é terminar com a maior pontuação possível antes do tempo acabar ou antes do risco explodir.

## Status

Primeira versão funcional criada em HTML, CSS e JavaScript puros.

## Como jogar localmente

Basta abrir o arquivo `index.html` no navegador.

Não precisa instalar dependências, rodar terminal, configurar banco de dados ou criar conta.

## Estrutura

```txt
index.html
src/
  main.js
  styles.css
```

## Funcionalidades

- Tempo limite de 45 segundos
- Até 10 rodadas
- Sistema de pontuação
- Sistema de risco
- Sistema de combo
- Recorde salvo no navegador com `localStorage`
- Layout responsivo
- Visual neon/dark
